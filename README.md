<h2>🧐 Features</h2>

Here're some of the project's best features:

*   You can add multiple bank accounts
*   make transactions from accounts
*   track your transaction history across accounts

<h2>🛠️ Installation Steps:</h2>

<p>1. BACKEND</p>

```
cd Backend
```

```
npm install
```

```
node server.js
```

<p>2. FRONTEND</p>

```
cd Frontend
```

```
npm install
```

```
npm run dev
```
